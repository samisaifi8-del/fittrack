# FitTrack

Personal weight-loss / muscle-gain tracker (single-page PWA).

- 5-day Push/Pull/Legs gym split with progressive-overload + PR tracking
- Calorie & macro tracking, weight trend, streaks, achievements
- Per-day history via calendar; all data stored locally on your device

## Run
```
npm start   # serves on $PORT (default 3000)
```
Deployed on Railway. Add to your phone home screen for an app-like experience.
